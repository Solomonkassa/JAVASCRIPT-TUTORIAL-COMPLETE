# Fundamentals: The Building Blocks

Master the core concepts that everything else builds upon.

## Topics Covered

1. **[Variables and Data Types](./01-variables-types.md)** - var, let, const, primitive types, objects
2. **[Operators](./02-operators.md)** - Arithmetic, logical, comparison, assignment, special operators
3. **[Type Conversion](./03-type-conversion.md)** - Implicit and explicit type coercion
4. **[Strings](./04-strings.md)** - String methods, manipulation, and operations
5. **[Error Handling](./05-error-handling.md)** - Try/catch, error types, throwing errors

## Learning Path

```
Variables & Types → Operators → Type Conversion 
→ Strings → Error Handling → Ready for Control Flow!
```

## Quick Reference

| Concept | What | Why |
|---------|------|-----|
| Variables | Store data | Reuse and manage values |
| Types | Data categories | Different operations |
| Operators | Perform actions | Manipulate data |
| Strings | Text data | Handle text information |
