# Getting Started with JavaScript

## What is JavaScript?

JavaScript is a versatile, dynamic programming language that powers interactive web experiences. It runs in browsers, servers (Node.js), and many other environments.

### Key Facts:
- **Created**: 1995 by Brendan Eich
- **Type**: Dynamically typed, interpreted language
- **Use Cases**: Web development, backend (Node.js), mobile apps, desktop apps, automation
- **Popularity**: Most popular programming language worldwide

## Why Learn JavaScript?

1. **Essential for Web Development** - The only language that runs natively in browsers
2. **Full-Stack Capable** - Use it for frontend AND backend
3. **Rich Ecosystem** - Thousands of libraries and frameworks
4. **High Demand** - Most sought-after programming skill
5. **Versatile** - Not limited to web - use for automation, data analysis, etc.

## Setting Up Your Environment

### Option 1: Browser Console (Easiest)
1. Open any browser (Chrome, Firefox, Safari, Edge)
2. Press `F12` or `Ctrl+Shift+I` (Windows) / `Cmd+Option+I` (Mac)
3. Click the "Console" tab
4. Start typing JavaScript!

```javascript
// Your first JavaScript code!
console.log("Hello, JavaScript!");
alert("Welcome to the JavaScript Journey!");
```

### Option 2: Node.js (For Backend)
1. Download from https://nodejs.org/
2. Install it
3. Create a file `hello.js`:
```javascript
console.log("Hello from Node.js!");
```
4. Run: `node hello.js`

### Option 3: Online Editors
- **CodePen**: https://codepen.io
- **JSFiddle**: https://jsfiddle.net
- **Replit**: https://replit.com
- **Glitch**: https://glitch.com

## Your First Program

### In Browser Console:
```javascript
// Example 1: Print to console
console.log("Hello, World!");

// Example 2: Display an alert
alert("This is JavaScript!");

// Example 3: Print multiple values
console.log("Learning", "JavaScript", "is", "fun!");
```

### In Node.js:
```javascript
// Create file: welcome.js
console.log("Welcome to JavaScript!");
console.log("This code runs on a server!");
```

## Understanding Output Methods

```javascript
// 1. console.log() - Most common, prints to console
console.log("This is a log message");

// 2. alert() - Shows a popup in browser
alert("This is an alert!");

// 3. console.error() - Prints error messages
console.error("This is an error");

// 4. console.warn() - Prints warning messages
console.warn("This is a warning");

// 5. document.write() - Writes to web page (use in HTML)
document.write("This appears on the page");
```

## Running Code Along the Way

3. **Run it** and observe the output
